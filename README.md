# d3lab
### Project Setup
Assuming you have `bower` already installed:
```
bower install
```

Now download the sample data:
```
curl -O https://s3.amazonaws.com/yubowenok/car.csv
```
### Usage
Start the webserver:
```
python -m SimpleHTTPServer
```

Then access your webpage by:
```
http://localhost:8000/
```
